# 🚀 Smart Spam Filter & Text Recommendation System

An NLP-based machine learning project that classifies messages as Spam
or Ham and provides next-word recommendations using a Bigram language
model.

## 📌 Features

-   Spam vs Ham classification
-   Text preprocessing (cleaning, tokenization, stemming)
-   Binary / Count / TF-IDF vectorization
-   Logistic Regression models
-   Bigram word recommendation system
-   Hugging Face deployed demo
